# 术语库 · Terminology (zh-CN ↔ en)

用于中英文案对齐，避免直译歧义。实现上以 `src/locales/*.json` 的 `terminology.*` 与界面文案为准；本表供产品/翻译对照。

| 概念 | 中文（站内用语） | English (site) | 说明 |
|------|------------------|----------------|------|
| 攻略 | 穷游攻略 / 攻略 | **travel guide** / **guide** | 可复用的行程、预算与贴士类内容（编辑精选或网友分享），**不是** generic “article” 或模糊的 “strategy”。 |
| 投稿 | 投稿 / 发布（网友） | **submit** (content) / **submission** | 用户向本站提交原创路线或经历；英文避免单独用易歧义的 “contribute” 作按钮主文案。 |
| 审核 | 审核 / 审核中 | **review** / **moderation** / **under review** | 公开展示前的合规与质量核查；**不用** “audit”（偏财务/内审语感）。 |

## 路由与键名

- URL 仍可为 `/articles`（历史原因）；展示层优先使用 `common.articles`（中文「穷游攻略」/ 英文 **Guides**）。
